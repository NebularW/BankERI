### 1.2 text

**功能**：获取指定外规的文本内容

**url**：`'/external/text'`

**入参**：name    (外规名称)

**返回值**：外规文本



### 1.3 cal

**功能**：获取与指定外规相似的内规

**url**：`'/cal'`

**入参**：name    (外规名称)

**返回值**：10个最相似的内规（包括其相似度名称、类别、文本）
